Juego de Adriana — Guerrera Femenina
-----------------------------------
Versión creada especialmente: estética femenina (rosa/lila/dorado), corazones como recolectables,
nubes/cristales como plataformas, picos como trampas y enemigos que disparan proyectiles.

Archivos incluidos:
- index.html
- style.css
- game.js
- README.txt

Cómo usar:
1. Descarga y descomprime el ZIP.
2. Abre index.html con el navegador (doble clic).
3. Usa las teclas: ← → ↑ Espacio (atacar) y P para pausar.

Si quieres: añadir sonidos, iconos PNG/SVG para la guerrera, música de fondo o subirlo a un host,
pídemelo y lo incorporo en la siguiente versión.
